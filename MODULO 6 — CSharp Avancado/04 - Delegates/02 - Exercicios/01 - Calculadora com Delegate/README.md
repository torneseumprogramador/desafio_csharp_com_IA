# 01 - Calculadora com Delegate

## Enunciado

Crie uma aplicação de console em C# que utilize `delegate` para representar operações matemáticas entre dois números inteiros.

O objetivo é praticar a criação de um delegate customizado e associá-lo a diferentes métodos.

## Requisitos

- Criar um delegate que receba dois números inteiros e retorne um número inteiro.
- Criar pelo menos os métodos `Somar` e `Subtrair`.
- Associar o delegate a cada um desses métodos.
- Executar as operações e exibir os resultados no console.
- Mostrar claramente qual operação foi executada.

## Objetivo da prática

- Entender a sintaxe de criação de um delegate.
- Aprender a apontar um delegate para métodos compatíveis.
- Praticar reaproveitamento de lógica com delegates.

## Desafio extra

- Adicionar as operações de multiplicação e divisão.
- Permitir que o usuário escolha qual operação deseja executar.
