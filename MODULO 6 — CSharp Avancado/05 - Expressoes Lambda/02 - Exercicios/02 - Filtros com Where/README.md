# 02 - Filtros com Where

## Enunciado

Crie uma aplicação de console em C# que trabalhe com uma lista de números inteiros e utilize expressões lambda para filtrar os dados.

O foco deste exercício é praticar o uso de lambdas com `Where` e entender como a lógica de filtro pode ficar mais simples e direta.

## Requisitos

- Criar uma lista de números inteiros.
- Usar `Where` com expressão lambda para filtrar apenas os números pares.
- Usar `Where` com expressão lambda para filtrar apenas os números ímpares.
- Exibir os resultados dos dois filtros no console.
- Criar também um método tradicional para verificar se o número é par.
- Repetir pelo menos um dos filtros usando esse método tradicional.

## Objetivo da prática

- Praticar o uso de lambda com `Where`.
- Comparar lambda com função tradicional em filtros.
- Entender como escrever expressões booleanas de forma curta.

## Desafio extra

- Criar filtros para números maiores que 5.
- Criar filtros para números múltiplos de 3.
