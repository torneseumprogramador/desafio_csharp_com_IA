# 01 - Funcoes com Lambda

## Enunciado

Crie uma aplicação de console em C# para praticar expressões lambda com os tipos `Func`, `Action` e `Predicate`.

O objetivo é transformar operações simples em lambdas e comparar esse estilo com funções tradicionais.

## Requisitos

- Criar uma lambda com `Func<int, int, int>` para somar dois números.
- Criar uma lambda com `Func<double, double, double>` para calcular a média de dois números.
- Criar uma lambda com `Func<string, string>` para gerar uma saudação a partir de um nome.
- Criar uma lambda com `Action<string>` para exibir uma mensagem no console.
- Criar uma lambda com `Predicate<int>` para verificar se um número é par.
- Executar todas as lambdas no `Main` e mostrar os resultados na tela.

## Objetivo da prática

- Entender quando usar `Func`, `Action` e `Predicate`.
- Praticar a sintaxe básica de expressões lambda.
- Comparar lambdas com métodos tradicionais.

## Desafio extra

- Criar uma lambda sem parâmetro usando `Action`.
- Reescrever uma das lambdas usando bloco com chaves.
