## Exercício 03 – Boas Práticas de Nomenclatura e Tipos

**Objetivo:** praticar boas práticas de nomes (PascalCase, camelCase, snake_case) e o uso de tipos estáticos em C#.

### Instruções

- **1. Crie um novo projeto console** em C# chamado `exercicio03_boas_praticas`.
- **2. Crie uma classe** chamada `UsuarioComDadosAnonimos` (use **PascalCase** para o nome da classe).
- **3. Dentro do `Main` (ou em outro ponto do programa), declare variáveis usando:**
  - `camelCase` para nomes de variáveis, por exemplo: `string usuarioComDadosAnonimos = "...";`
  - `snake_case` apenas como *exemplo em `string`*, por exemplo:  
    `string usuario_com_dados_anonimos = "usuario_com_dados_anonimos";`
- **4. Mostre no console** cada uma dessas variáveis para visualizar os diferentes estilos de nome.
- **5. Adicione comentários explicando** (em português) em qual situação você usaria:
  - PascalCase
  - camelCase
  - snake_case

### Regras

- Use **tipos estáticos** (`string`, `int`, etc.) nas declarações.
- Não use recursos ainda não estudados (sem `if`, `while`, etc.).

### Desafio extra (opcional)

- Crie mais **duas variáveis** seguindo o padrão:
  - Uma para representar a idade de um usuário em `camelCase`.
  - Uma para representar o e-mail de um usuário em `camelCase`.
- Mostre essas informações no console usando `Console.WriteLine`.
